# PROJETO-DE-EXTENSAO-EM-DESENVOLVIMENTO-DE-SOFTWARE
Projeto acadêmico voltado à aplicação prática de HTML, C e Banco de Dados, evidenciando conhecimentos em desenvolvimento de software, lógica de programação e modelagem de dados.

sistema que cadastra produtos manualmente, atualiza seu estoque automaticamente, e se mantem atualizado em posiçao de produto (em qual etapa o produto esta, sendo essas, estoque, pedido, baixa de estoque)

uso simples. ao começar o programa deve-se escolher uma opçao.

cadastro, seguir os passos que serao pedidos pelo sistema, para cadastrar tanto o produto quanto suas especificaçoes

consulta de estoque total, e por produto

movimentação de estoque, podendo adicionar novos itens no estoque ou dar baixa manual de um pedido
